## Node.js: 
Wird verwendet, um JavaScript außerhalb des Browsers auszuführen. Es ermöglicht die Entwicklung von serverseitigen Anwendungen und Tools.

## npm (Node package manager): 
Ein Paketmanager , der alle Pakete und Abhängigkeiten für Node.js verwaltet. Es ermöglicht die einfache Installation und Verwaltung von Bibliotheken und Tools.

## Vite: 
Ein Buildtool, um die Entwicklung von Webanwendungen zu beschleunigen. Die Daten werden schneller geladen und die Entwicklungserfahrung verbessert.

## Warum Vite für den Anfang praktisch ist: 
Weil man dirkekt loslegen kann, ohne viel Konfiguration. Es bietet eine schnelle Entwicklungsumgebung und ermöglicht es dir, dich auf das Schreiben von Code zu konzentrieren, anstatt Zeit mit dem Einrichten zu verbringen.
